# Basic_Assignment
## Overview
Native Android App built with Jetpack Compose, Ktor Client, Exoplayer and Supabase to demonstrate video playback.
## Features
* Search Videos
* Play Videos
## Tech Stack Used
* Kotlin
* jetpack Compose
* Ktor
* PostgreSQL
* Supabase
* MVVM
